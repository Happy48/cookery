| api  | method  | 说明  | 返回格式  |

|  /api/search?key=xxx[&page=1] | get  |  根据关键词进行搜索食品 | [{"foodTitle":"xxx","foodPic":"xxx(url)","foodDesc":"xxx","foodLikes":1,"foodCreateTime":"xxx","foodShare"分享的作品:"xxx","foodUrl":"xxx"}]  |
